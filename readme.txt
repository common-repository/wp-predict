WP-Predict lets you create questions for your readers on a per post basis and then rank your readers in a sidebar leaderboard.

== Description ==

WP-Predict enables you to add multiple choice questions to your blog posts. Your readers can then select an answer. Once you set the answer your readers will be ranked in a leaderboard widget. 

To see screenshots of the plugin visit http://www.pootlepress.co.uk/wordpress-plugins/

== Installation ==

1. Upload `wp-predict.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to settings | wp-predict
4. Select a post
5. Post some prediction quesions.


== Frequently Asked Questions ==
How do I use the plugin?
Once activated, in your dashboard a new menu item will appear under settings called wp-predict. 
Click on wp-predict, then select which post you want to add a question to. Fill in the details and publish.
